Features

Object Avoidance

Our device uses a custom object detection pipeline built with OpenCV and MobileNet-SSD to identify nearby obstacles in real time. When a potential hazard is detected, the system delivers immediate voice feedback to alert the user, enabling them to safely navigate their environment without needing visual input.

Voice Controls

Using Google's Speech Recognition library, users can control the device hands-free through simple voice commands. This allows for smooth interaction with core functions such as object detection, reading text, and accessing the AI chatbot, significantly enhancing independence and ease of use.

Connectivity

The device offers seamless connectivity with smartphones and other smart devices, ensuring updates and interactions can occur wirelessly. This allows users or caregivers to sync data, adjust settings, or integrate with additional assistive technologies with minimal friction.

AI Chatbot

Powered by DialoGPT, our AI chatbot enables users to have natural, helpful conversations with the device. It guides them through different functions, answers questions, and improves accessibility by offering a friendly, conversational interface that makes the experience more intuitive.

360° Vision

360° vision is a planned future enhancement that will utilize an array of cameras to give users full environmental awareness. This feature will further improve safety and orientation by offering real-time alerts from all directions, helping users avoid hazards from behind or side angles.

Augmented Reality

Augmented Reality integration is part of our future roadmap. It aims to overlay contextual information into the user's auditory space, providing directional cues, object descriptions, or location-based alerts, thereby transforming how visually impaired users perceive and navigate the world.

Reading

To assist with reading printed or digital text, the device uses OCR technology via Tesseract to detect and extract text, which is then converted into speech using Microsoft Azure’s advanced Text-to-Speech service. This enables users to listen to books, signs, and labels instantly and independently.
